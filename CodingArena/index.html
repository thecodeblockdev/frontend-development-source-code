What is HTML?

HTML stands for HyperText Markup Language, it's a markup language developed by sir Tim-Berners Lee in 1993 to develop web pages

HTML consists of elemets that describes different type of content: Headings, images, links, paragraphs, buttons, forms, tables, videos etc.
